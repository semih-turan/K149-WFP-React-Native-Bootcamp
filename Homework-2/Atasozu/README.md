### Second Week Homework 

For install:

`` npm install --legacy-peer-deps ``

Return to ios file:

`` cd ./Atasozu/ios ``

For ios, you need to install:
`` pod install ``

Return back to project root file:

`` cd ./Atasozu ``

Start Metro Server:

`` npm start run ``

For ios emulator: 

`` npm run ios `` 

For android emulor:

`` npm run android ``
